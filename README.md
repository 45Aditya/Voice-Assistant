 ## Voice Assistant

A Virtual Assistant developed using PYTHON-3


## Specification

It will help you to perform following task:

1- Weather Forecast

2- News Headlines

3- Search Wikipedia

4- Search Location

5- Open Application

6- Can tell Joke
